# What is this?

This repo aims to declare the patterns of the Synaptic Standard, an alternative
to the Dendritic Pattern. The Synaptic Standard aims to solve many of the same
problems as the Dendritic Pattern, but without a ton of boilerplate, or a
reliance on `flake-parts`.

The reason I'm writing this is because I think the dendritic pattern gets all
the PROBLEMS right - it just solves them in the wrong ways. This guide is split
into three parts, which should be read in order:

1. [What are the problems with a typical NixOS configuration?](./docs/first.md)

2. [How does the synaptic standard solve these problems?](./docs/second.md)

3. [How is the synaptic standard "better" than the dendritic pattern (at least
   according to me)?](./docs/third.md)
